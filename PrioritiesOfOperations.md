### Priorities of operations in java

| Operators            | Precedence                          |
|----------------------|-------------------------------------|
| postfix              | expr++ expr--                       |
| unary                | ++expr --expr +expr -expr ~ !       |
| multiplicative       | * / %                               |
| additive             | + -                                 |
| shift                | << >> >>>                           |
| relational           | < > <= >= instanceof                |
| equality             | == !=                               |
| bitwise AND          | &                                   |
| bitwise exclusive OR | ^                                   |
| bitwise inclusive OR | I                                   |
| logical AND          | &&                                  |
| logical OR           | II                                  |                                   |	                               |
| ternary              | ? :                                 |
| assignment           | = += -= *= /= %= &= ^= <<= >>= >>>= |
